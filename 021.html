ประวัติส่วนตัว:
ชื่อ นางสาว ศริณพร โนนทิง
ชื่อเล่น เชียร์
เกิดวันที่ 13 มกราคม 2548
สัญชาติ ไทย
อายุ 20ปี
ที่อยู่ 59/39 จังหวัดนนทบุรี อำเภอเมืองนนทบุรี ถนนนนทบุรี ตำบลท่าทราย 11000
ประวัติการศึกษา:
• ระดับประกาศนียบัตรวิชาชีพ: สำเร็จจากการศึกษาจาก โรงเรียนนนทบุรีวิทยาลัย
• ระดับประริญยาตรี: กำลังศึกษาอยู่ที่ มหาวิทยาลัยเทคโนโลยีราชมงคลสุวรรณภูมิ
• สาขา วิทยาการคอมพิวเตอร์
ทักษะ:

การเขียนโปรแกรมและพัฒนาเว็บไซต์

• เข้าใจพื้นฐาน HTML, CSS
• Database
• python
• เข้าใจพื้นฐาน UX/UI
• ศิลปะและการออกแบบ การวาดรูป งานศิลปะ
• ค้นคว้าและวิเคราะห์

คุณสมบัติ

• พร้อมเรียนรู้เทคโนโลยีใหม่ๆ
• มีความสนใจพื้นฐานด้านคอมพิวเตอร์ และเทคโนโลยีสารสนเทศ

เป้าหมายในอนาคต

• เป้าหมายระยะสั้น: สำเร็จการศึกษาระดับปริญญาตรี และพัฒนาทักษะการเขียนโปรแกรมให้มีความชำนาญ
• เป้าหมายระยะยาว: ทำงานในสายงานเทคโนโลยีสารสนเทศและมีส่วนร่วมใหการพัฒนานวัตกรรมที่เป็นประโยชน์ต่อสังคม
ผลงาน:
• แสดงข้อความ บนหน้าจอ LCD ผ่านบอร์ด ESP32 โดยควบคุมผ่านปุ่มในแอป Blynk IoT บนมือถือ
• ออกแบบฐานข้อมูลพื้นฐาน Database การทำงานของไอศครีม
• ออกแบบ UX/UI แอปพลิเคชันเพื่อเรียนรู้สำหรับเด็กเล็ก
